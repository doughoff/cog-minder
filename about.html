Cog-Minder is a helper website for Cogmind. It has a parts searcher that takes information from the in-game gallery CSV export and turns it into this searchable webpage, a bots bestiary that lists all bots in the game and their stats, a build planner that allows for creating a build loadout and viewing stats for it, a hacking calculator that shows hacking success rates, a combat simulator that predicts combat outcomes from a collection of equipped parts, and a RIF reference for RIF abilities and bot hacking. It should have all items and bots as of Beta 11, let me know if anything seems missing.

This is an MIT licensed open source project with the source available on GitHub.
